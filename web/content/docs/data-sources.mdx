whoami.wiki agents can work with many types of personal data. The more sources you provide, the richer and more accurate your encyclopedia pages become.

## Supported formats

Agents can process most common file formats:

- **Images**: JPEG, PNG, HEIC, WebP, TIFF
- **Video**: MP4, MOV, AVI
- **Text**: JSON, CSV, TXT, HTML
- **Archives**: ZIP (will be extracted automatically)

## Data export guides

Most platforms let you export your data. Here are some common ones:

### Google

- **Google Photos**: [Google Takeout](https://takeout.google.com/) — select "Google Photos"
- **Google Maps Timeline**: Google Takeout — select "Location History"
- **Gmail**: Google Takeout — select "Mail"

### Social media

- **Facebook**: Settings → Your Information → Download Your Information
- **Instagram**: Settings → Your Activity → Download Your Information
- **WhatsApp**: Settings → Chats → Export Chat (per conversation)

### Financial

- Most banks offer CSV exports of transaction history from their web dashboard

### Music

- **Shazam**: Settings → My Music → Export as CSV
- **Spotify**: Request data from Privacy Settings

## How agents use your data

When given multiple data sources, agents cross-reference them to build richer narratives. For example, an agent processing a trip might:

1. Use photos for visual context and timestamps
2. Use location history for precise routes and durations
3. Use bank transactions to identify restaurants and venues
4. Use messages for personal context and conversations that happened during the trip
