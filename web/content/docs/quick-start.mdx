description: "Create your first encyclopedia page in minutes"
---

This guide walks you through creating your first encyclopedia page from a folder of photos.

## Prepare your data

Create a directory with the photos you want to write about. For best results, start with a focused set — a single trip, event, or time period.

```bash
mkdir ~/whoami-data/family-trip-2023
# Copy your photos into this directory
```

## Create a page

Point the CLI at your data directory:

```bash
whoami write --source ~/whoami-data/family-trip-2023
```

The agent will:

1. Browse through your photos, creating contact sheets for efficient viewing
2. Extract EXIF metadata (dates, locations, camera info)
3. Identify people, places, and events from visual content
4. Draft a Wikipedia-style page with sections, captions, and linked references

## Review and refine

The draft will be created as a new page in your local MediaWiki instance. Open `http://localhost:8080` to review it.

You can refine the page by providing additional context:

```bash
whoami refine --page "Family Trip 2023" --context "The tall man in the blue shirt is my uncle David"
```

The agent will incorporate your notes and update the page accordingly.

## Add more data sources

For richer pages, you can point the agent at multiple data sources:

```bash
whoami write --source ~/whoami-data/family-trip-2023 \
  --source ~/exports/google-maps-timeline.json \
  --source ~/exports/bank-transactions.csv
```

The agent will cross-reference across all sources — matching photos to locations, transactions to restaurants, and timelines to events.
