---
title: Citation System
description: "How citations link wiki content back to source material"
---

Citations are how wiki pages reference the underlying source data. Every factual claim in a page should link back to a specific piece of evidence in the archive.

## How citations work

Citations use MediaWiki's `<ref>` syntax with custom templates that identify the source type and archive location. When rendered, they appear as numbered footnotes with links back to the source material.

## Citation templates

### Messages

For citing a specific message or conversation:

```wikitext
<ref>{{cite message
  |platform=WhatsApp
  |participants=Alice, Bob
  |date=2023-03-15
  |hash=abc123def456
}}</ref>
```

Renders as a footnote linking to the archived message, showing the platform, participants, and date.

### Voice notes

For citing audio recordings:

```wikitext
<ref>{{cite voice
  |speaker=Alice
  |date=2023-06-20
  |duration=2:34
  |hash=def789ghi012
}}</ref>
```

### Photos

For citing a specific photograph:

```wikitext
<ref>{{cite photo
  |date=2023-07-04
  |location=Central Park, New York
  |hash=ghi345jkl678
}}</ref>
```

Renders with a thumbnail preview of the photo and its metadata.

### Videos

For citing video recordings:

```wikitext
<ref>{{cite video
  |date=2023-12-25
  |duration=1:15:30
  |hash=jkl901mno234
}}</ref>
```

## Archive hashes

Each citation includes a `hash` field — the content-addressed identifier of the source file in the archive. This ensures citations point to a specific, immutable piece of data. Even if the original file is moved or renamed, the citation still resolves.

## Footnotes section

Pages include a footnotes section at the bottom that collects all citations:

```wikitext
== References ==
{{reflist}}
```

This renders as a numbered list of all sources cited in the page, similar to a Wikipedia references section.
